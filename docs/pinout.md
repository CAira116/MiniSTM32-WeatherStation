# MiniSTM32 V3.0 气象站引脚分配表

## 主控芯片：STM32F103RCT6 (LQFP64, 72MHz)

## 引脚分配

| 功能模块     | 引脚   | 说明                            |
|-------------|--------|---------------------------------|
| **DS18B20** | PG11   | 单总线温度传感器 (OneWire)        |
| **OLED SCL**| PB6    | I2C1 时钟线 (SSD1306 128×64)     |
| **OLED SDA**| PB7    | I2C1 数据线                      |
| **24C02 SCL**| PB6   | 与 OLED 共用 I2C1 总线           |
| **24C02 SDA**| PB7   | 与 OLED 共用 I2C1 总线           |
| **USART1 TX**| PA9   | 串口发送 → CH340G → USB → PC    |
| **USART1 RX**| PA10  | 串口接收 ← CH340G                |
| **LED0 (DS0)**| PE5  | 红色 LED，低电平点亮              |
| **LED1 (DS1)**| PB5  | 绿色 LED，低电平点亮              |
| **KEY0**    | PE4    | 用户按键，按下为低电平             |
| **KEY1**    | PE3    | 用户按键，按下为低电平             |
| **WK_UP**   | PA0    | 唤醒按键，按下为高电平             |

## I2C 设备地址

| 设备        | I2C 地址        | 说明               |
|------------|-----------------|--------------------|
| SSD1306    | 0x3C (0x78>>1) | OLED 显示屏         |
| 24C02      | 0x50 (0xA0>>1) | EEPROM 256 字节    |

## 硬件连接确认清单

- [ ] DS18B20 插入板上的 DS18B20 接口（PG11）
- [ ] OLED 模块连接到 I2C 排针（PB6/PB7）
- [ ] USB 线插入 USB1 口（CH340G 串口）
- [ ] 跳线帽：BOOT0=0, BOOT1=0（从 Flash 启动）
- [ ] 电源开关打开

> ⚠️ 实际引脚请以你的原理图为准，可通过 `strings` 命令或 PDF 查看器确认。
